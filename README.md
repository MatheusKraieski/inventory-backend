# inventory-backend

## Project setup

```
pip install django djangorestframework
```

### Pillow

```
python -m pip install Pillow
```

### Extensions

```
pip install django-extensions
```

### Run server

```
python manage.py runserver
```

### MPTT 

```
pip install django-mptt
```

### Heroku
```
pip install django-heroku
```

### Gunicorn
```
pip install gunicorn
```
